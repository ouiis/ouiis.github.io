CSS 文字單位（px、em、rem…）

發表於 2020-06-08 分類於 CSS

CSS 的文字單位分為絕對單位與相對單位。絕對單位是直接指定該元素的大小；相對單位則是會視裝置大小、根元素（html）或父元素的大小而定。

絕對單位

  • px：螢幕顯示的單位（pixel），組成螢幕圖像的最小單位。是最常使用的單位，瀏覽器預設的字體大小為 16px。

  • pt：很少使用的單位，也是印刷的單位（point），另一種是 iOS 所定義的 pt。

CSS 設計模式 - OOCSS

發表於 2020-05-28 分類於 CSS

OOCSS 是 Object Oriented CSS 的縮寫，由 Nicole Sullivan 所提出的 CSS 設計模式，而這裡指的物件是可以在網站上重複使用的 CSS 樣式。

OOCSS 有兩個大原則：

  1. Separate Structure and Skin（結構與樣式的分離）
  2. Separate Container and Content（容器與內容的分離）

結構與樣式的分離

結構指的是無法直接用肉眼看出來的部分，例如元素被指定的大小和位置。

  • Height
  • Width
  • Margins
  • Padding
  • Overflow

樣式指的是元素可見的部分，例如：

  • Colors
  • Fonts
  • Shadows
  • Gradients

CSS 設計模式 - SMACSS

發表於 2020-05-27 分類於 CSS

在小型專案裡，由於架構並不複雜，在管理上較不容易出現問題，但隨著專案的規模的成長，或者團隊人數的增加，如何讓程式碼的架構一目瞭然，方便自己或其他開發者維護專案，成為一個必須思考的課題。

SMACSS 是 Scalable and Modular Architecture for CSS 的縮寫，由 Jonathan Snook 提出的 CSS 設計方法，目的在於使 CSS 的結構具有延展性以及模組化。

CSS 設計模式 - BEM

發表於 2020-05-23 分類於 CSS

BEM (Block Element Modifier) 是由 Yandex 所提出的 CSS 設計模式，一種 CSS class 命名方式，如名稱所示，命名是由「區塊、元素、修飾符」所組成。

BEM 文件

Block

Block 是頁面上每一個獨立的元件，以區塊功能來命名 class，不以區塊外觀命名。

  • block 可獨立在頁面上任何地方，因此每個 block 都不應被位置限制住，不要在 block 寫 margin、position 等影響外部環境樣式。
  • 使用 BEM 設計模式時不應使用標籤、ID 選擇器。
  • 可以和其他 block 巢狀包覆。
