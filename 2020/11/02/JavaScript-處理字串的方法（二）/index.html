JavaScript 處理字串的方法（二）

發表於 2020-11-02 分類於 JavaScript

要將一段字串中的大寫轉換成小寫時，可以使用字串方法 toLowerCase()，這篇將紀錄不使用 JavaScript 內建的方法，要如何實作相同的結果。

  • 將字串轉成陣列，以分析每個字元。
  • 跑迴圈將字元轉成 Unicode 編碼。
  • 將大寫編碼（65 到 90）加上 32 變成小寫編碼（97 到 122）。
  • 將轉換完成的編碼結果新增到陣列。
  • 最後將陣列轉換成字串回傳。

在程式語言中處理文字時，有時候會碰到 Unicode。Unicode 是一種標準，它對世界上大部分的文字系統進行了整理和編碼，使電腦可以透過更簡單的方法呈現或處理文字。其中與 Unicode 編碼相關的有 charCodeAt 和 fromCharCode() 這兩個方法。

charCodeAt()

  • 從字串中取得第 n 個字元的字元碼。
  • 帶入的引數為該字元索引。
  • 回傳值是 16 位元的整數，介於 0 到 65535 之間。
  • 若 n 是負的、大於或等於字串長度（找不到字元的索引值），會回傳 NaN。

fromCharCode()

  • 從字元編碼建立一個字串。
  • 可以傳入多個字元的 Unicode 編碼。
  • 回傳一個含有指定編碼字元的新字串。
  • 這是一個靜態方法，String() 建構式的一個特性，而非字串或 String 物件的方法。

toLowerCase()

回傳字串的副本，其中所有的字元都被轉為小寫（lowercase）。
